# 🚀 Nascimento Solutions - Landing Page de Alta Performance

Uma landing page moderna, impactante e estratégica para a **Nascimento Solutions**, focada em Performance Digital, ROI e soluções de escala para negócios B2B.

Desenvolvida com foco em autoridade visual, carregamento instantâneo e conversão agressiva de leads.

### 🎯 Funcionalidades Principais

- **Design Brutalista Moderno**: Identidade visual forte com cores neon e tipografia imponente.
- **Seções Estratégicas**: Detalhamento de Serviços, Metodologia de Trabalho, Prova Social e FAQ.
- **Mockups Interativos**: Simulação de interface de alta conversão com mapas de calor e métricas de ROI.
- **Menu Hambúrguer Responsivo**: Navegação fluida em todos os dispositivos.
- **Botão Flutuante de WhatsApp**: Canal direto de vendas integrado com mensagem personalizada.
- **Otimização de Performance**: Código leve garantindo carregamento rápido (ideal para mobile).
- **Smooth Scroll & Back to Top**: Experiência de navegação suave e intuitiva.

### 💻 Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Tailwind CSS** (estilização moderna e responsiva)
- **Framer Motion** (animações premium e transições de seção)
- **Lucide React** (ícones premium)
